# Дипломная работа "Бэкенд-часть сайта объявлений на Java"
<h2>Описание</h2>

Сайт объявлений о продаже вещей с комментариями.
Реализована бэкенд часть проекта платформы по перепродаже вещей, позволяющая пользователям размещать объявления товаров и оставлять комментарии к другим объявлениям.

<h2>Что реализовано в бэкенд-части проекта?</h2>

Авторизация и аутентификация пользователей.

Распределение ролей между пользователями - администратор и пользователь.

CRUD для объявлений на сайте: администратору даны полномочия удалять или редактировать все объявления, а пользователям — только свои.

Под каждым объявлением пользователи могут оставлять отзывы.

В заголовке сайта можно осуществлять поиск объявлений по названию.

Показывать и сохранять картинки объявлений.
